
📘 AUDIO SPEED CONTROL SETUP (NO CODE REQUIRED)

✅ What it does:
Lets students hear audio at different speeds (slow 🐢, normal 🎧, fast 🚀)

✅ How to use:
1. Add this code to any section layout where Admin wants speed controls
2. Example use: Dictionary page, Spelling page, Dictation practice

✅ How Admin chooses where it appears:
In layout editor, Admin can:
☑ Add speed control block to any section
☑ Pick default speed (0.25, 0.5, 1)
☑ Turn off during tests

✅ You can add it to:
- Dictionary.jsx
- Dictation.jsx
- SpellingDictation.jsx
- Any section block defined by Admin
